## v1.10.0 更新日志（PGMM / Power Grid Minimap）

### 🧩 电网显示过滤（高级）
- 🧹 新增高级设置项「忽略面积小于 X 的电网」（单位：格²）。
  - 电网面积 = 有效发电厂/用电建筑/存电建筑的面积总和。
  - 当面积 < X 时，该电网会在电网表与小/大地图上被忽略显示；设为 0 表示关闭。
