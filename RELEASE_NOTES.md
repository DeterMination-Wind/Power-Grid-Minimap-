## v1.9.0 更新日志（PGMM / Power Grid Minimap）

### 🗺️ 电网小地图 / 大地图叠加层
- 🎨 为每个**断开的电网**单独着色（小地图 + 全屏大地图）。
- 🔢 在电网中心显示**电力盈亏**（可调字号/颜色/透明度）。
- 🧩 对“激光/远距离连接导致的稀疏电网”支持**分块聚类**：远离的分块会分别标记中心，近的分块合并为一个中心标记。
- 🧭 支持全屏大地图的平移/缩放对齐（叠加层跟随大地图视野）。

### 🚨 断网告警与建议连接点
- ⚠️ 大电网分裂后，若在时间窗内出现负电，会提示并标记**建议连接点**与连线。
- 🎚️ 阈值/时间窗/线宽/颜色均可在设置中调整。

### 🛟 缺电救援建议（测试/Beta）
- 🛟 当电网持续负电时，尝试圈出建议隔离的“正电岛”，帮助先恢复部分电网运行。
- 🧨 支持“激进模式”（扫描更多候选链接，可能更吵但覆盖更广）。

### 🪟 电网电力表（Power Table）
- 📋 显示高于阈值的电网：`#电网ID`、`in`、`now`、`min`。
- 🖱️ 在 MindustryX 的 **OverlayUI** 窗口中，点击 `#电网ID` 可将视角移动到该电网中心。
- 🧲 优先集成 MindustryX OverlayUI（可拖拽/固定/缩放）；未安装时自动回退到普通 HUD 侧边显示。

### 🔄 自动检测更新（GitHub）
- 🌐 启动游戏后异步检查 GitHub 是否有新版本（优先读取 Releases，失败回退到仓库 `mod.json`）。
- 🧾 发现更新时弹窗提示，可一键打开 Release 页面；也可“忽略此版本”。
- ⏱️ 内置检查间隔（避免频繁请求）。
- ⚙️ 设置中可开关“检查更新/弹窗提示”。

### ⚙️ 设置与构建
- 🧰 设置界面采用 MindustryX 风格组件（图标 + 分组 + 统一样式）。
- 📦 构建产物：同时输出 `.zip` / `.jar`（并复制到 `dist/`）。
- 🗂️ 本地构建后额外复制到根目录 `构建/`，文件名为 `powergrid-minimap-版本号.*`，便于分发与归档。
